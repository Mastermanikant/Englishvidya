# 13_Threat_Model.md

## संभावित हमले के परिदृश्य (हैकर्स के दिमाग से)

| # | हमले का प्रकार | विवरण | लक्ष्य | संभावित प्रभाव |
|---|----------------|--------|--------|-----------------|
| 1 | **SQL Injection** (यदि कोई सर्वर‑साइड DB को सीधे क्वेरी करता है) | असमानित इनपुट को `?` या `;--` के साथ भेजना | D1/D2 टेबल (`usage_log`, `rewards` आदि) | डेटा चोरी/बदलाव, उपयोग‑स्टैट्स मिटना |
| 2 | **Cross‑Site Scripting (XSS)** | यूज़र कमेंट/फ़ीडबैक फ़ॉर्म में `<script>` इन्जेक्ट करना | सभी फ्रंट‑एंड पेज | सत्र हाईजैक, क्रेडेंशियल चोरी, मलबा UI |
| 3 | **Cross‑Site Request Forgery (CSRF)** | लॉगर‑इन यूज़र के ब्राउज़र से दुश्मन साइट के माध्यम से `/api/usage` या `/api/reward` को ट्रिगर करना | सत्र‑आधारित API | फर्जी उपयोग‑रेकोर्ड, पॉइंट‑फ़्रॉड |
| 4 | **Clickjacking / UI Redress** | `iframe` के भीतर साइट लोड करके यूज़र को नकली बटन पर क्लिक करवाना | विज्ञापन/रिवॉर्ड बटन | फर्जी ad‑click, रिवॉर्ड दुरुपयोग |
| 5 | **Credential Stuffing / Brute Force** | Google Sign‑In या Email/Password (यदि fallback) पर बार‑बार लॉगिन प्रयास | Auth सिस्टम | अकाउंट लॉक, बॉट‑लॉगिन |
| 6 | **Man‑in‑the‑Middle (MITM)** | असुरक्षित HTTP (यदि किसी कस्टम API को HTTP पर चलाते) | सभी API कॉल | डेटा स्निफ़, टोकन चोरी |
| 7 | **Denial‑of‑Service (DoS) / Resource Exhaustion** | अत्यधिक `/api/usage` राइट या `/api/comment` स्पैम | Workers/D1 | क्वोटा परिनति, साइट स्लो डाउन |
| 8 | **Monetag/Ad‑Network Fraud** | बॉट्स द्वारा वीडियो एड‑इम्प्रेशन बनाना | Monetag एपीआई | विज्ञापन बिल्ड‑अप, राजस्व चोरी |
| 9 | **Referrer Abuse** | रेफ़रल कोड को कई डिवाइस/IP से पुनः‑उपयोग | रेफ़रल सिस्टम | बेतरतीब इनकम, फ़्रॉड |
|10| **Data Leakage via Mis‑Configured CORS** | सार्वजनिक CORS हेडर से `*` अनुमति देना | API 엔드포인트 | डेटा एक्सफ़िल्ट्रेशन |
|11| **Local Storage Tampering** | `localStorage` या `sessionStorage` में टाइम/डिवाइस‑ID बदलना | क्लाइंट‑साइड ट्रैकिंग | फर्जी उपयोग‑स्टैट्स |
|12| **Service Worker Cache Poisoning** | कैश में पुरानी/मैलिशियस स्क्रिप्ट डालना | PWA | खराब UX / मैलिशियस कोड डिप्लॉय |
|13| **Voice Search Injection** (यदि Voice API जोड़ते) | ऑडियो इनपुट को हाइजैक कर कमांड एग्ज़ीक्यूट करना | Voice फीचर | अनुप्रयुक्त कमांड चलना |

**नोट:** अधिकांश हमले क्लाइंट‑साइड या Cloudflare Workers पर केंद्रित हैं क्योंकि हमारी आर्किटेक्चर सर्वर‑लेस है।
